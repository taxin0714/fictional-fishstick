# fictional-fishstick
环境科学AI实验助手

中文 | English （可选，如果你需要英文版可以后续添加）

一个面向环境科研工作者、学生和监测人员的AI智能体，旨在通过自然语言交互，极大降低多源异构环境数据的处理、分析与可视化门槛。

🌟 项目简介

无论是撰写论文、完成大创项目，还是进行日常环境监测，您是否也曾被海量数据困扰？不同来源且结构复杂的数据，往往导致清洗难、分析慢、绘图繁琐等问题，大量时间被消耗在重复性操作上，而非聚焦于科学问题本身。

环境科学AI实验助手 应运而生。它允许您像与领域专家对话一样，使用自然语言指令，即可完成复杂的环境数据分析流程，并自动生成符合学术规范的专业图表与解读。

✨ 核心特性

🤖 自然语言交互：无需编写代码，直接用中文描述您的分析需求（例如：“分析温度与湿度的相关性”、“绘制近十年降水量时间序列图”）。

🌍 多场景支持：全面支持水质、气候、土壤、森林火险等多种环境数据类型。

📊 智能图表生成：自动生成箱线图、时间序列图、热力图、散点图等专业级图表，质量可达发表级别。

专业解读：基于环境科学规范，提供准确的统计结果和风险评估建议。

🛠 技术架构

本系统基于以下技术栈构建：

大语言模型：智谱AI GLM-4，负责理解用户意图并规划分析流程。

交互层：使用 Gradio 框架构建简洁易用的Web界面。

计算与可视化：通过提示工程调用 Pandas、NumPy、Matplotlib、Seaborn 等Python科学计算库，确保分析结果的准确性和专业性。

🚀 快速开始

前提条件

Python 3.8+

安装必要的Python包

安装步骤

克隆仓库

git clone https://github.com/你的用户名/你的仓库名.git cd 你的仓库名 

安装依赖

pip install -r requirements.txt 

配置API密钥

复制 config.example.py 文件并重命名为 config.py。

在其中填入您的智谱AI API密钥。

cp config.example.py config.py # 然后编辑 config.py 文件 

运行应用

python app.py 

运行后，终端将显示一个本地URL（通常是 http://127.0.0.1:7860），在浏览器中打开即可使用。

访问方式

本地访问：运行上述命令后，在浏览器中打开本地URL。

公网访问：我们集成了 localtunnel 支持，运行后会自动生成一个公共URL，方便团队协作和移动端访问。

📖 使用示例

准备数据：准备一个CSV格式的环境数据文件（例如：森林火险数据）。

上传数据：在工具界面中上传您的CSV文件。

下达指令：在输入框中用自然语言提出分析需求，例如：

“分析温度与湿度的相关性。”

“绘制不同月份降水量的箱线图。”

“展示土壤含水率随温度和降水变化的趋势。”

获取结果：系统将自动执行分析，并生成包含统计结果、可视化图表和专业文字概述的报告。

降低技术门槛：使学生和科研人员能专注于专业判断，而非数据处理技术细节。
